Telegram Bot для компании по остеклению

Этот проект представляет собой Telegram бота для компании, занимающейся остеклением окон и балконов. Бот помогает собирать заявки от клиентов через интерактивное меню и сохранять данные о заказе.

Функциональность

Интерактивное меню с состояниями (Finite State Machine)
Сбор информации о типе остекления, размерах, контактных данных
Портфолио и контактная информация компании
Сводка заказа с отправкой данных клиенту
Подготовка к интеграции с Google Таблицами для сохранения заявок


Основные компоненты

Finite State Machine (FSM) - управление диалоговыми состояниями
UserSession - управление сессиями пользователей
Состояния бота - определены в словаре fsm
Ключевые состояния

main_menu - главное меню
glazing_* - состояния для остекления окон
balcony_* - состояния для остекления балконов
contacts - контактная информация
portfolio - портфолио работ
adress, phone, name - сбор контактных данных
summary - итоговая сводка заказа


Пользователь запускает бота командой /start
Проходит через интерактивное меню, выбирая нужные опции
Вводит контактные данные
Получает сводку заказа
Данные сохраняются (при настроенной интеграции)
